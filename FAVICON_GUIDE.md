# Favicon Setup Guide

## Quick Setup

1. **Generate Favicons** (Easiest Method):
   - Go to [RealFaviconGenerator](https://realfavicongenerator.net/)
   - Upload your logo or create a simple design
   - Download the generated files
   - Place all files in `/public/` folder

2. **Required Files**:
   ```
   /public/
   ├── favicon.ico
   ├── favicon-16x16.png
   ├── favicon-32x32.png
   ├── apple-touch-icon.png (180x180)
   ├── android-chrome-192x192.png
   └── android-chrome-512x512.png
   ```

3. **Simple Design Option**:
   - Use your initials "DB" in a circle
   - Use your brand colors (purple #4C1D95 and gold #F59E0B)
   - Create a simple, recognizable icon

4. **Open Graph Image**:
   - Create `/public/images/og-image.jpg`
   - Size: 1200x630px
   - Should include your name, tagline, and a professional background
   - This appears when sharing your portfolio on social media

## Design Tips

- Keep it simple and recognizable at small sizes
- Use high contrast colors
- Test on different backgrounds
- Make sure it works in both light and dark themes
